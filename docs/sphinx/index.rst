dsbuilder: creating datasets, to specification
==============================================

**dsbuilder** is a python module to help you build and write data files (primarily `NetCDF <https://www.unidata.ucar.edu/software/netcdf/>`_) with a specified format of dimensions, variables and metadata. This is built on top of `xarray <http://xarray.pydata.org/en/stable/>`_.
